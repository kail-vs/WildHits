Wild Hits is a minimalist 2D arcade shooting game with old‑school charm and modern polish.  
Zip around the levels, fire at targets, and push your reflexes and patience to the limit across multiple challenges.

---

## Controls

Take command with a few simple keys:

| Key | Action |
|-----|--------|
| **S** | Start Level |
| **P** | Pause Game |
| **R** | Resume After Pause |
| **A** | Restart Level |
| **N** | Go to Next Level |
| **Q** | Quit to Main Menu |

Aim carefully and think on your feet — bullets and seconds are precious.

---

## Behind the Scenes

Wild Hits is crafted with care using a lightweight 2D game framework that handles graphics and input, giving a smooth, responsive feel honed for web and desktop play.  
The game logic, level flow, and UI all come together to emulate classic arcade gameplay while being accessible right in your browser.
